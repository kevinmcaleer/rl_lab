I want to create a reinforcement learning simulation environment, where users can experiment, an learn how reinforcement learning works. The environment should use existing 3d simulation tools such as rviz or foxglove. To begin with lets use the Buddy Jr robot arm (https://www.kevsrobots.com/blog/buddy_jr.html) so a 3d udrf file for that model will need to be created. 
Produce a number of experiments for the user to try, which culminate in a greater understanding of RL that they can use in their own robotics projects.
